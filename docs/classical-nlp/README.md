# Classical NLP

## About This Section

This section covers pre-deep-learning NLP methods that dominated the field before neural approaches. While modern NLP relies heavily on neural networks and transformers, classical methods remain important for understanding NLP fundamentals, providing baselines, and being practical in resource-constrained settings.

Classical NLP teaches you to think about language through statistical patterns, count-based features, and probabilistic models. These intuitions carry forward even when working with modern systems -- understanding TF-IDF helps you understand retrieval, n-grams explain language modeling basics, and topic modeling relates to semantic understanding.

## Contents

### [N-grams and Language Modeling](ngrams-and-lm.md)

Statistical language modeling using n-grams. Covers bigrams, trigrams, probability estimation, smoothing techniques (Laplace, Good-Turing, Kneser-Ney), perplexity as an evaluation metric, and the limitations of count-based approaches. N-grams provide intuition for what language models do and why longer context matters.

### [Vector Space Models](vector-space-models.md)

Representing text as vectors for similarity and retrieval. Covers bag-of-words representation, TF-IDF weighting, document similarity (cosine similarity), and applications in information retrieval and text classification. These methods underpin many modern retrieval systems and provide baselines for semantic similarity.

### [Topic Modeling](topic-modeling.md)

Discovering latent topics in document collections. Covers Latent Dirichlet Allocation (LDA), Latent Semantic Analysis (LSA), model interpretation, choosing the number of topics, and practical applications. Topic modeling reveals document-level themes and provides dimensionality reduction for text.

### [Classical Text Classification](classical-classification.md)

Traditional machine learning for text classification. Covers feature engineering, Naive Bayes classifiers, logistic regression, support vector machines, and evaluation. Understanding these methods builds intuition for classification tasks that transfers to modern approaches.
