# Fundamentals

## About This Section

This section covers the foundational building blocks of natural language processing. Before working with embeddings, language models, or LLMs, you need to understand how text is processed at a fundamental level: how we break text into units, prepare it for analysis, identify linguistic structures, and extract meaningful information.

These fundamentals apply across all NLP approaches -- whether you're building classical statistical models or working with modern transformers. Understanding these concepts helps you make informed decisions about text processing pipelines, debug issues, and appreciate what modern models learn automatically.

## Contents

### [Tokenization](tokenization.md)

Breaking text into meaningful units (tokens). Covers word tokenization, subword tokenization, character tokenization, and the trade-offs between different approaches. Understanding tokenization is critical because it determines what units your model operates on and affects vocabulary size, OOV handling, and multilingual support.

### [Text Preprocessing](text-preprocessing.md)

Preparing raw text for NLP tasks. Covers lowercasing, stemming, lemmatization, stopword removal, normalization, and when to apply each technique. Different tasks require different preprocessing -- understanding the trade-offs helps you make informed choices about what to keep and what to discard.

### [Linguistic Foundations](linguistic-foundations.md)

Core linguistic concepts that underpin NLP. Covers part-of-speech tagging, named entity recognition, parsing, and fundamental linguistic concepts. These provide the vocabulary and frameworks for understanding how language works and what NLP systems try to capture.
